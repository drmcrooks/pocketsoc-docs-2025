# Access

- There are 15 nodes available; credentials will be made available along with central node URL.

- Each training node is accessible through a set of ports for each service with schema:

    - https://CENTRAL_URL:4XX0Y
    - XX: client
    - Y: Service
        - 1: Portainer
        - 2: OpenSearch
        - 3: MISP
